## libreta-ui

The frontend for libreta. Written in SolidJS with tailwindcss.

Demo with `npm run dev`.

## Structure

`src/`               - index.tsx, App.tsx, and css files.
`src/assets/`        - static assets.
`src/components/`    - tsx components.
`public/`            - static files used globally.
`index.html`         - frontend entrypoint.
`package.json`       - npm package file.
`tsconfig.json`      - typescript config.
`tsconfig.app.json`  - typescript config.
`tsconfig.node.json` - typescript config.
`vite.config.ts`     - vite config.
